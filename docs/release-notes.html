Version 1.3.86 November 29, 2018


  • AMRSDK 1.3.86    ~ New delegate methods added to AMRInterstitialDelegate, AMRRewardedVideoDelegate and AMROfferWallDelegate.
  • Admob 7.36.0.0    ~ Native banner ad format changed with GADUnifiedNativeAd.
  • Adtiming 3.0.8.0
  • Facebook 5.1.0.1    ~ Native banner ads updated.
  • Error codes added to AMRError.h file. 
  • - (void)didFailToShowRewardedVideo:(AMRRewardedVideo *)rewardedVideo;
  • New delegate method added to AMRInterstitialDelegate protocol to catch if interstitial failed to be shown; 
    - (void)didFailToShowInterstitial:(AMRInterstitial *)interstitial error:(AMRError *)error;
  • New delegate method added to AMRRewardedVideoDelegate protocol to catch if rewarded video failed to be shown; 
    - (void)didFailToShowRewardedVideo:(AMRRewardedVideo *)rewardedVideo error:(AMRError *)error;
  • New delegate method added to AMROfferWallDelegate protocol to catch if offerwall failed to be shown; 
    - (void)didFailToShowOfferWall:(AMROfferWall *)offerwall error:(AMRError *)error;
  • New delegate method added to AMROfferWallDelegate protocol to catch if offerwall shown; 
    - (void)didShowOfferWall:(AMROfferWall *)offerwall;
